# Rybelsus Guide

- **Type:** Static HTML blog
- **Domain:** rybelsusguide.org
- **Stack:** HTML/CSS/JS (no build step)

## What This Is

Evidence-based guides on Rybelsus (oral semaglutide) for weight loss. Covers dosage protocols, side effects, clinical trial results.

## Structure

- `index.html` — homepage
- `posts/` — article pages
- `style.css` / `script.js` — styling and behavior
- `wrangler.jsonc` — Cloudflare deployment config

## Deployment

Cloudflare Pages. Deploy with `wrangler deploy`.
